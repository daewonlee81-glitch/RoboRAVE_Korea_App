배포 상태

모바일 배포에 필요한 기본 구성이 이미 포함되어 있습니다.

포함된 항목
PWA Manifest Service Worker 오프라인 캐시 홈 화면 설치
  • 한 폴더만 올리면 되는 정적 사이트 구조입니다.
  • `https://` 환경에서 설치형 앱처럼 동작합니다.
  • 로컬에서는 `start.command`로 `http://127.0.0.1` 서버를 띄웁니다.
권장 배포 방법
  1. 이 폴더 전체를 Netlify Drop, GitHub Pages, Vercel 같은 정적 호스팅에 업로드합니다.
  2. 모바일에서 배포 주소를 열고 브라우저의 홈 화면 추가 기능을 사용합니다.
  3. 현장 운영용이면 아이폰과 안드로이드에서 각각 한 번씩 설치 동작을 확인합니다.

배포 루트는 현재 폴더 하나면 충분합니다. 순위표와 대진표는 모두 같은 앱 안에서 이동할 수 있도록 정리되어 있습니다.
